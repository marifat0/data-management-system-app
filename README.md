# data-management-system-app

Tracing Data Alumni PIA

## Tutorial Menjalankan
Pertama : Clone github repo ini kedalam local directory

Kedua : Jalankan command dibawah di dalam terminal untuk menginstall modul modul sudah tertera di package.json:
```
npm install
```
Ketiga : Jalankan command berikut untuk menjalankan localhost
```
npm run dev
```

> Pastikan sudah menginstall nodejs dan memastikan command NPM telah berjalan dengan baik sebelum menjalankan perintah-perintah diatas

Cara memastikan NPM dan Nodejs sudah baik adalah dengan menjalankan perintah berikut:
```
npm -v
node -v
```
